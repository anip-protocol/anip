# GitHub Governed Fronting Showcase

Generated by `anip-generate --target python` from an exported `anip-service-definition.json`.

## What is generated

- FastAPI-based ANIP host using `anip-service` and `anip-fastapi`
- generated capability declarations and runtime routing
- explicit backend adapter and policy seams for handwritten completion
- smoke tests covering discovery and first-capability invocation for every contract service

## Services

- `github-governance-service`: `python -m github_governed_fronting_showcase.app`

## Commands

- `python -m github_governed_fronting_showcase.app`
- `pytest`

## Live GitHub Behavior

- `github.repo.search_context` runs bounded repository-scoped issue/PR search.
- `github.issue.prepare` returns a create-issue preview from live GitHub repository metadata.
- GitHub issue creation is disabled unless `ANIP_GITHUB_ALLOW_MUTATION=true` and the invoke
  carries a runtime-validated ANIP `approval_grant`.
