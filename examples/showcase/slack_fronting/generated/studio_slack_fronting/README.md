# Slack Governed Fronting Showcase

Generated by `anip-generate --target python` from an exported `anip-service-definition.json`.

## What is generated

- FastAPI-based ANIP host using `anip-service` and `anip-fastapi`
- generated capability declarations and runtime routing
- explicit backend adapter and policy seams for handwritten completion
- smoke tests covering discovery and first-capability invocation for every contract service

## Services

- `slack-governance-service`: `python -m slack_governed_fronting_showcase.app`

## Commands

- `python -m slack_governed_fronting_showcase.app`
- `pytest`

## Live Slack Behavior

- `slack.channel.read_context` runs bounded channel-history reads.
- `slack.thread.summarize` runs bounded thread reply reads.
- `slack.message.prepare` returns a message preview from declared input.
- Slack posting is disabled unless `ANIP_SLACK_ALLOW_SEND=true` and the invoke
  carries a runtime-validated ANIP `approval_grant`.
