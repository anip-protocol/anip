# GTM Showcase Developer Runbooks

This directory holds repo-local operational documentation for the GTM showcase.

Use the public docs site for narrative explanation:

```text
website/docs/showcases/gtm-agent/
```

Use this directory when you need exact file paths, commands, and release-maintenance checklists.

## Runbooks

- `local-stack.md`: ports, compose files, images, and local startup commands.
- `release-artifacts.md`: package, bundle, generated output, image, and snapshot locations.
- `question-bank.md`: where the 540-case GTM benchmark suite, 24 hard-mode governance cases, and run artifacts live.
