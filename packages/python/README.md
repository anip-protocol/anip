# Python Packages

This workspace contains the Python ANIP runtime and interface packages.

Generated services include local backend adapter seams and fronting backend
templates for downstream REST, GraphQL, MCP, SQL, semantic-query, or provider SDK
calls. ANIP does not ship unreleased outbound adapter helper packages as a
required dependency layer.

Use normal Python SDKs, HTTP clients, MCP clients, database clients, or
organization-owned libraries inside the generated backend seam.
