# Version History

ANIP is moving in public, and the version line tells the story of what became first-class over time.

## Current line

**v0.11** is the current protocol/runtime line.

It adds:

- runtime observability
- logging, metrics, tracing, and diagnostics hooks
- `getHealth()` and optional health endpoint support

## Recent versions

| Version | Main addition |
| --- | --- |
| `v0.11` | Runtime observability |
| `v0.10` | Horizontal scaling support |
| `v0.9` | Audit aggregation |
| `v0.8` | Security hardening |
| `v0.7` | Discovery posture |
| `v0.6` | Streaming invocations |

## Why include version history on the site

ANIP is evolving quickly, and the site should make that progression legible:

- what exists today
- what stabilized recently
- what is still future work

For the longer roadmap, see the main spec and repo docs.
